# Squirrel's Desk

A daily planning tool that pulls from three sources: the previous day's Obsidian 
debrief, the week's Google Calendar events, and the current Todoist task list.

Before surfacing the plan it asks two questions: what is your energy level today, 
and how many hours do you actually have. The output is scoped to reality, not to 
a full calendar of theoretical availability.

Surfaces carry-over items, upcoming meetings, and open tasks into a single 
daily HTML briefing file. The plan lives in the file, not in a chat thread, 
so it does not disappear when the session ends.
