# Walking Pad Workout Tool

A browser-based workout tracker built for a treadmill desk. Single HTML file, no backend, no login. A live countdown timer drives the full session: warmup, circuits, and cool down across a 5-day weekly structure. Equipment swap options are built into each exercise. Belt-off phases for lower body day are treated as a distinct phase type with explicit cues, not a footnote.

Built to solve a specific workflow problem: eliminating the pre-session decisions that break routines before they start.
